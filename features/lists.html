HTML Lists

HTML supports ordered (<ol>) and unordered (<ul>) lists.

Unordered List

<ul> <li>First item</li> <li>Second item</li> <li>Third item</li> </ul>
  • First item
  • Second item
  • Third item

Ordered List

<ol> <li>First step</li> <li>Second step</li> <li>Third step</li> </ol>
  1. First step
  2. Second step
  3. Third step

Nested Lists

<ul> <li>Main item 1 <ul> <li>Sub item</li> <li>Sub item</li> </ul> </li> <li>Main item 2</li> </ul>
  • Main item 1
    • Sub item
    • Sub item
  • Main item 2
