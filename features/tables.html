HTML Tables

Tables are created with <table>, <tr> (rows), <th> (headers), and <td> (data cells).

<table border="1"> <caption>Monthly Savings</caption> <tr> <th>Month</th> <th>Savings</th> </tr> <tr> <td>January</td> <td>$100</td> </tr> <tr> <td>February</td> <td>$150</td> </tr> </table>
Monthly Savings
Month Savings
January $100
February $150

Advanced Table Features

<table border="1"> <thead> <tr> <th colspan="2">Header spanning 2 columns</th> </tr> </thead> <tbody> <tr> <td rowspan="2">Cell spanning 2 rows</td> <td>Row 1, Cell 2</td> </tr> <tr> <td>Row 2, Cell 2</td> </tr> </tbody> <tfoot> <tr> <td>Footer 1</td> <td>Footer 2</td> </tr> </tfoot> </table>
Header spanning 2 columns
Cell spanning 2 rows Row 1, Cell 2
Row 2, Cell 2
Footer 1 Footer 2
Note: While the border attribute works, modern tables should be styled with CSS.
